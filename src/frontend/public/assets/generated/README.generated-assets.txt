Generated Assets - Price Tag Stickers
======================================

This directory contains generated vintage-style price tag sticker assets.

Available assets:
- /assets/generated/price-tag-mint.dim_512x512.png
- /assets/generated/price-tag-near-mint.dim_512x512.png
- /assets/generated/price-tag-played.dim_512x512.png

Each sticker is 512x512px with a transparent background, featuring:
- Vintage paper texture
- Small peeling corner effect
- Color-coded text (MINT=green, NEAR MINT=blue, PLAYED=orange)
